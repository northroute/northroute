## 🧩 Project Directions

### 🔬 Science Info Platform

A university-oriented platform for collecting and processing scientific publication data.

**Focus areas:**

- Scopus / Web of Science parsing;
- author and affiliation matching;
- h-index / ORCID data;
- publication normalization;
- admin tools for manual verification;
- background jobs and parsing status tracking.

**Stack:** `.NET 8`, `PostgreSQL`, `Playwright`, `Docker`, `GitLab CI`

---

### 📚 Library & University Automation

Internal tools for library workflows, data migration, reporting, and system integration.

**Focus areas:**

- Koha / IRBIS-like systems;
- MARC data;
- book availability and educational supply cards;
- document generation;
- internal dashboards.

**Stack:** `.NET`, `SQL`, `Koha`, `Docker`, `Word/OpenXML`

---

### 🌐 AirpathVPN

A small VPN subscription service with payments, user cabinet, admin panel, and deployment infrastructure.

**Focus areas:**

- user cabinet;
- subscription logic;
- payment integration;
- admin dashboard;
- server nodes;
- Docker + Traefik deployment.

**Stack:** `ASP.NET Core`, `Razor Pages`, `Identity`, `PostgreSQL`, `Docker`, `Traefik`

---

### 🎮 English Learning Game Prototype

A Godot Mono prototype where English skills are connected to game mechanics.

**Idea:**

- Reading gives information;
- Writing can deal damage;
- Listening helps with navigation;
- Speaking affects NPC interactions.

**Stack:** `Godot Mono`, `C#`, `local LLM experiments`
